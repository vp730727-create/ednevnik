# 📚 Е-Дневник — Инструкции за инсталация

## Какво ви трябва
- Компютър с Windows / Mac / Linux
- Интернет връзка

---

## СТЪПКА 1 — Инсталирайте Node.js

1. Отидете на **https://nodejs.org**
2. Свалете версията с надпис **"LTS"** (препоръчана)
3. Инсталирайте я (просто Next → Next → Finish)
4. Проверете дали работи: отворете **Command Prompt** (Windows) или **Terminal** (Mac) и напишете:
   ```
   node --version
   ```
   Трябва да видите нещо като `v20.11.0`

---

## СТЪПКА 2 — Настройте Gmail за изпращане на имейли

> Нужно е за да получавате имейл когато някой се регистрира.

1. Влезте в акаунта **faketeacherconfirmation@gmail.com**
2. Отидете на **https://myaccount.google.com/security**
3. Включете **"2-Step Verification"** (ако не е включено)
4. След това отидете на **https://myaccount.google.com/apppasswords**
5. В полето "App name" напишете `Ednevnik` и натиснете **Create**
6. Ще получите парола от 16 символа (напр. `abcd efgh ijkl mnop`) — **запазете я!**

---

## СТЪПКА 3 — Настройте файла .env

Отворете файла **`.env`** в папката на проекта с Notepad и попълнете:

```
GMAIL_USER=faketeacherconfirmation@gmail.com
GMAIL_PASS=abcdefghijklmnop        ← паролата от стъпка 2 (без интервали)
ADMIN_EMAIL=fragments@abv.bg
SESSION_SECRET=напишете_произволни_букви_и_цифри_тук_123abc
APP_URL=http://localhost:3000
PORT=3000
```

Запазете файла.

---

## СТЪПКА 4 — Стартирайте сървъра

Отворете **Command Prompt / Terminal**, отидете в папката на проекта:

```bash
cd path/to/ednevnik
```

Инсталирайте пакетите (само веднъж):
```bash
npm install
```

Стартирайте сървъра:
```bash
npm start
```

Трябва да видите:
```
✅ Е-Дневник работи на http://localhost:3000
```

Отворете браузъра и отидете на **http://localhost:3000** 🎉

## Как работи системата

### Регистрация на нов учител:
1. Учителят отива на `/signup` и попълва името, имейла и паролата си
2. Системата изпраща имейл до `fragments@abv.bg` с бутон **"Одобри потребителя"**
3. Кликате бутона в имейла → учителят е одобрен
4. Учителят получава имейл че може да влезе
5. Учителят влиза с имейла и паролата си на `/login`

### Ако учителят чака одобрение:
- Вижда страница "Изчаква одобрение" и не може да достъпи дневника

---

## Качване онлайн (по желание)

За да е достъпен от интернет, качете го на **Railway.app** (безплатно):

1. Създайте акаунт на **https://railway.app**
2. Натиснете **"New Project"** → **"Deploy from GitHub"**
3. Качете файловете в GitHub и свържете репото
4. В Railway добавете Environment Variables (от `.env` файла)
5. Сменете `APP_URL` с реалния URL от Railway (напр. `https://ednevnik-production.up.railway.app`)

## Файлова структура

```
ednevnik/
├── server.js          ← главен файл, стартирайте него
├── db.js              ← база данни
├── email.js           ← изпращане на имейли
├── .env               ← настройки (не качвайте в интернет!)
├── package.json       ← пакети
├── routes/
│   ├── auth.js        ← регистрация, вход, одобрение
│   └── app.js         ← дневникът
├── views/
│   └── app.html       ← дневникът (HTML)
└── middleware/
    └── auth.js        ← проверка дали е влязъл
```
